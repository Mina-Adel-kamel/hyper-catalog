أنت مهندس برمجيات Full Stack ومصمم واجهات محترف.

أريد إنشاء موقع ويب كامل يسمى:

"هايبر براند"

وهو عبارة عن نظام لإنشاء مجلة عروض السوبر ماركت مثل مجلات كارفور.

يجب أن يتم بناء المشروع باستخدام:

HTML
CSS
JavaScript (Vanilla)

بدون استخدام أي Framework.

يجب أن يكون التصميم احترافي وسهل الاستخدام وقابل للطباعة.

------------------------------------------------

هدف النظام

النظام يسمح لصاحب المحل بـ:

• إدارة المنتجات
• إضافة منتجات جديدة
• رفع صور المنتجات
• تعديل الأسعار
• حذف المنتجات
• تنظيم المنتجات حسب الأقسام
• إنشاء مجلة عروض
• تصدير المجلة كملف PDF للطباعة

------------------------------------------------

1) غلاف المجلة

يجب إنشاء صفحة غلاف احترافية للمجلة.

يحتوي الغلاف على:

العنوان الرئيسي:
"هايبر براند"

عنوان فرعي:
"مجلة عروض السوبر ماركت"

خلفية تحتوي على صورة سوبر ماركت أو منتجات غذائية.

خط كبير وواضح مثل مجلات كارفور.

يجب أن تكون هذه الصفحة هي الصفحة الأولى عند طباعة المجلة.

------------------------------------------------

2) لوحة التحكم (Admin Panel)

يجب إنشاء لوحة تحكم تسمح للمستخدم بإدارة المنتجات.

الوظائف المطلوبة:

إضافة منتج
تعديل منتج
حذف منتج
رفع صورة المنتج
تعديل السعر
تعديل الوصف

حقول إضافة المنتج:

اسم المنتج
القسم
الوصف
السعر
رفع صورة المنتج

الأزرار:

إضافة المنتج
تحديث المنتج
حذف المنتج

------------------------------------------------

3) هيكل المنتج

كل منتج يجب أن يحتوي على البيانات التالية:

id
name
category
description
price
image

------------------------------------------------

4) الأقسام

يجب تقسيم المنتجات إلى أقسام مثل:

الزيوت
الأرز
المكرونة
السكر
المعلبات
الألبان
السناكس
المشروبات
المنظفات
العناية الشخصية

كل قسم يظهر في صفحة خاصة به داخل المجلة.

------------------------------------------------

5) عرض المنتجات

يجب عرض المنتجات داخل كروت (Product Cards).

كل كارت يحتوي على:

صورة المنتج
اسم المنتج
الوصف
السعر

يجب أن يكون التصميم مثل كتالوج أو مجلة عروض.

------------------------------------------------

6) البحث

يجب إضافة خاصية البحث عن المنتجات.

يمكن البحث باستخدام:

اسم المنتج
القسم

ويتم تحديث النتائج مباشرة أثناء الكتابة.

------------------------------------------------

7) حذف المنتج

يجب أن يحتوي كل منتج على زر:

حذف

وعند الضغط يتم حذف المنتج من المجلة.

------------------------------------------------

8) حفظ البيانات

يجب حفظ المنتجات باستخدام:

LocalStorage

حتى لا تختفي البيانات عند إعادة تحميل الصفحة.

------------------------------------------------

9) تصميم المجلة للطباعة

يجب تصميم المجلة بحيث تكون مناسبة للطباعة.

المواصفات:

صفحات بحجم A4
كل صفحة تحتوي على 4 أو 6 منتجات
فواصل صفحات بين الأقسام
الغلاف في الصفحة الأولى

------------------------------------------------

10) تصدير المجلة PDF

يجب إضافة زر:

تحميل المجلة PDF

عند الضغط عليه يتم إنشاء ملف PDF يحتوي على:

الغلاف
جميع الأقسام
جميع المنتجات

باستخدام مكتبة:

html2pdf.js

------------------------------------------------

11) تصميم الواجهة

يجب أن يكون التصميم:

نظيف
احترافي
يشبه مجلات السوبر ماركت
Responsive
